## 可靠性、可维护性、可用性
### 可靠性度量
平均无故障时间(Mean Time To Failure)
平均失效间隔(Mean Time Between Failure)
### 可维护性度量
平均维修时间(Mean Time To Repair)
### 可用性
MTTF / ( MTTF + MTTR ) * 100%
### 可靠性(reliability) 可用性(availability) 区别

## 串联、并联、混联系统
串联：R = R1 * R2 * ... * Rn
并联：R = 1 - (1 - R1) * (1 - R2) * ... * (1 - Rn)
混联系统：R * (1 - (1 - R)^3) * (1 - (1 - R)^2)
